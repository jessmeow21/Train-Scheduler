# Train-Scheduler
##Homework for week 7/Firebase

##About:
This web application lets users enter their information on any train and returns information on the next timed arrival and the frequency of the train. 

Using:
*HTML
*CSS
*Bootstrap
*Javascript
*JQuery
*Firebase
*Moment.js
